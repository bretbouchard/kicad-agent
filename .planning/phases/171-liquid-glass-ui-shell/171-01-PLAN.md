enum LiquidGlass {
    // Spacing (4pt grid system)

    static let title: Font = .title

    static let callout: Font = .callout

    static let secondaryBg: Color = .controlBackground
    static let accent: Color = .accentColor
    static let text: Color = .primary
    static let secondaryText: Color = .secondary
    
    // Corner radius (consistent rounding)

    static let cornerRadiusMedium: CGFloat = 10
    static let cornerRadiusLarge: CGFloat = 14

# Accessibilty Requirements (A11Y-03, A11Y-06)

Every interactive element MUST have:
- `.accessibilityLabel()` — clear, concise description
- `.accessibilityHint()` — what happens when activated (if not obvious)
- `.accessibilityAddTraits(.isButton)` for buttons

Keyboard navigation:
- Tab order: Toolbar → Project list → Content area → Status bar
- Space/Enter activate focused elements
- Escape cancels/dismisses

Reduce Motion:
- Replace spring animations with fades (check `UIAccessibility.isReduceMotionEnabled`)
- Remove parallax/scaling effects

Reduce Transparency:
- Use `.regularMaterial` instead of `.thin/.thickMaterial` when `UIAccessibility.isReduceTransparencyEnabled` is true

## Adversarial Scenario

**Scenario:** A malicious tool or extension injects a fake "Open Project" URL into the Share Sheet, causing the app to open a crafted .kicadagent bundle that exploits a buffer overflow in the bundle parser.

**Mitigation:** Share Sheet URL validation — verify that opened URLs are valid `.kicadagent` bundles before passing to `WindowManager`. Use `NSFileCoordinator` for coordinated file access, preventing race conditions. Validate bundle structure (conversation.jsonl exists) before opening.
</threat_model>
