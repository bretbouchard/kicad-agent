    - `SpatialQueryEngine` class:
      ```python
      class SpatialQueryEngine:
          """Spatial query engine backed by Shapely STRtree.

          Builds a spatial index over extracted primitives for fast
          proximity, containment, and clearance queries.

          Uses two-phase query pattern:
          1. STRtree.query() for coarse bounding-box filter
          2. Exact Shapely intersection/contains/distance check on candidates
          """

          def __init__(self, primitives: list) -> None:
              """Build spatial index from a list of spatial primitives.

              Args:
                  primitives: List of SpatialPoint/SpatialBox/SpatialPath/SpatialRegion objects.
                  Empty list creates an empty engine (all queries return empty results).
              """
              self._primitives: list = list(primitives)
              if primitives:
                  geometries = [p.to_shapely() for p in primitives]
                  self._tree = STRtree(geometries)
              else:
                  self._tree = None

          @property
          def primitive_count(self) -> int:
              return len(self._primitives)
      ```

      - `proximity(self, x: float, y: float, radius_mm: float) -> list`:
        Find all primitives within radius_mm of point (x, y).
        1. Validate radius_mm > 0 (raise ValueError if not) and <= 10000.0 (DoS mitigation, T-08-08)
        2. Validate x, y are finite (not NaN/Inf)
        3. Create `query_point = shapely.Point(x, y)`
        4. Create `buffer = query_point.buffer(radius_mm)`
        5. If tree is None (empty), return []
        6. `candidates = self._tree.query(buffer)` -- returns array of indices
        7. Filter: `[self._primitives[i] for i in candidates if self._primitives[i].to_shapely().intersects(buffer)]`
        8. Return filtered list

      - `containment(self, x1: float, y1: float, x2: float, y2: float) -> list`:
        Find all primitives fully contained within the bounding box (x1,y1)-(x2,y2).
        1. Validate x1 < x2 and y1 < y2
        2. Validate coordinates are finite
        3. Create `query_box = shapely.box(x1, y1, x2, y2)`
        4. `candidates = self._tree.query(query_box)`
        5. Filter: `[self._primitives[i] for i in candidates if query_box.contains(self._primitives[i].to_shapely())]`
        6. Return filtered list

      - `clearance(self, entity_id: str, search_radius_mm: float = 10.0) -> list[tuple[Any, float]]`:
        Find all primitives near a given entity and compute distances.
        1. Find the primitive with matching entity_id (linear scan -- entity_id is not indexed)
        2. If not found, return []
        3. Create buffer around target geometry with search_radius_mm
        4. `candidates = self._tree.query(buffer)`
        5. For each candidate (excluding self): compute `target_geom.distance(candidate_geom)`
        6. Return `sorted([(primitive, distance), ...], key=lambda x: x[1])`

      - `find_by_entity_id(self, entity_id: str) -> list`:
        Find all primitives with matching entity_id. Returns list (may be empty or have multiple matches).

      - `find_by_net(self, net_name: str) -> list`:
        Find all primitives belonging to the named net. Checks `hasattr(p, 'net') and p.net == net_name`.

      - `find_by_layer(self, layer_name: str) -> list`:
        Find all primitives on the specified layer.

    All query methods return lists of the original primitive objects (not copies). The engine holds references to the input primitives list.

    - `SpatialViolation` frozen dataclass:
      ```python
      @dataclass(frozen=True)
      class SpatialViolation:
          """DRC/ERC violation with coordinate-grounded spatial context."""
          description: str
          severity: str  # "error", "warning", "exclusion"
          violation_type: str
          items: tuple[SpatialPoint, ...]  # Each violation item as a coordinate point
          spatial_context: str  # Human-readable spatial description with coordinates
          raw_items: tuple[dict, ...]  # Original item dicts from kicad-cli report

          def to_json(self) -> dict:
              return {
                  "description": self.description,
                  "severity": self.severity,
                  "violation_type": self.violation_type,
                  "items": [p.to_json() for p in self.items],
                  "spatial_context": self.spatial_context,
              }

          def format_report(self) -> str:
              """Format as coordinate-grounded report string.

              Example: 'The via at <point> [45.2, 22.1] violates clearance from <path> [...]'
              """
              parts = [f"[{self.severity.upper()}] {self.description}"]
              for item in self.items:
                  parts.append(f"  at <point> [{item.x:.4f}, {item.y:.4f}] ({item.entity_type})")
              parts.append(f"  {self.spatial_context}")
              return "\n".join(parts)
      ```

    - `enrich_drc_result(drc_result: DrcResult, pcb_ir: PcbIR | None = None) -> list[SpatialViolation]`:
      Convert DRC violations to spatially-grounded violations.
      1. For each violation in `drc_result.violations + drc_result.unconnected_items`:
         a. Extract coordinates from each violation item dict (items[i]["pos"]["x"], items[i]["pos"]["y"])
         b. Create SpatialPoint for each item with coordinates. Entity_type = "drc_item". Entity_id = item.get("uuid", f"drc_{i}")
         c. Build spatial_context string: "Violation at <point> [{x}, {y}] involving {len(items)} items"
         d. If pcb_ir provided, find nearest footprint to each violation coordinate (optional enrichment -- add "near component {ref}" to context)
      2. Return list of SpatialViolation objects
      3. If drc_result has error_message set, return empty list (no violations to enrich)

    - `enrich_erc_result(erc_result: ErcResult) -> list[SpatialViolation]`:
      Convert ERC violations to spatially-grounded violations. ERC items may have pos coordinates or may not (depends on violation type).
      1. Same pattern as enrich_drc_result but for ErcResult violations
      2. Handle items without pos data gracefully -- create SpatialPoint at (0.0, 0.0) with entity_type="erc_item_no_pos"

    - `_build_spatial_context(items: tuple[SpatialPoint, ...], nearby_info: str = "") -> str`:
      Format spatial context string from violation item points.
      ```python
      if len(items) == 0:
          return "No coordinate data available for this violation"
      elif len(items) == 1:
          p = items[0]
          base = f"Violation at <point> [{p.x:.4f}, {p.y:.4f}]"
      else:
          coords = ", ".join(f"[{p.x:.4f}, {p.y:.4f}]" for p in items)
          base = f"Violation involves {len(items)} items at positions: {coords}"
      if nearby_info:
          return f"{base}. {nearby_info}"
      return base
      ```

    - `_find_nearest_footprint(x: float, y: float, pcb_ir: PcbIR) -> str | None`:
      Find the nearest footprint reference to a coordinate. Returns reference designator string or None. Uses simple Euclidean distance to footprint position.
