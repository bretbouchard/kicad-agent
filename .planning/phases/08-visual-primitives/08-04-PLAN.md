    - `RickFinding` frozen dataclass:
      ```python
      @dataclass(frozen=True)
      class RickFinding:
          """Single coordinate-grounded finding from a Rick domain analysis."""
          domain: str              # "si", "pi", "emc", "dfm"
          category: str            # Domain-specific category (e.g., "crosstalk", "decoupling", "clearance")
          severity: str            # "critical", "warning", "info"
          description: str         # Human-readable finding description
          coordinates: tuple[tuple[float, float], ...]  # Affected coordinate positions
          affected_entities: tuple[dict, ...]  # JSON of nearby spatial primitives
          spatial_context: str     # Coordinate-grounded context string
          recommendation: str      # Fix recommendation

          def to_json(self) -> dict:
              return {
                  "domain": self.domain,
                  "category": self.category,
                  "severity": self.severity,
                  "description": self.description,
                  "coordinates": list(self.coordinates),
                  "affected_entities": list(self.affected_entities),
                  "spatial_context": self.spatial_context,
                  "recommendation": self.recommendation,
              }

          def format_finding(self) -> str:
              parts = [
                  f"[{self.domain.upper()}] [{self.severity.upper()}] {self.category}: {self.description}",
              ]
              for x, y in self.coordinates:
                  parts.append(f"  at <point> [{x:.4f}, {y:.4f}]")
              if self.affected_entities:
                  parts.append(f"  Affected entities: {len(self.affected_entities)} primitives nearby")
              parts.append(f"  {self.spatial_context}")
              parts.append(f"  Recommendation: {self.recommendation}")
              return "\n".join(parts)
      ```

    - `SpatialRickReport` frozen dataclass:
      ```python
      @dataclass(frozen=True)
      class SpatialRickReport:
          """Coordinate-grounded report for a specific Rick analysis domain."""
          domain: str
          board_path: str
          findings: tuple[RickFinding, ...]
          summary: str  # One-line summary of findings count and severity distribution

          def to_json(self) -> dict:
              return {
                  "domain": self.domain,
                  "board_path": self.board_path,
                  "findings": [f.to_json() for f in self.findings],
                  "summary": self.summary,
              }

          def format_report(self) -> str:
              lines = [f"=== {self.domain.upper()} Rick Report ===", self.summary, ""]
              for finding in self.findings:
                  lines.append(finding.format_finding())
                  lines.append("")
              return "\n".join(lines)
      ```

    - `generate_spatial_report(domain: RickDomain, pcb_primitives: list, spatial_violations: list[SpatialViolation] | None = None, board_path: str = "") -> SpatialRickReport`:
      Generate a coordinate-grounded report for a specific Rick domain.

      Implementation for each domain:

      **SI Rick (Signal Integrity):**
      - Category "crosstalk": Find traces (SpatialPath primitives) on the same layer that run parallel within 2mm. For each pair, create a finding with coordinates of the parallel segments. Severity = "warning" for parallel runs > 5mm, "critical" for > 10mm.
      - Category "trace_impedance": Find traces wider than 0.5mm or narrower than 0.1mm. Create findings for anomalous widths.
      - Category "stub_traces": Find traces (paths) that end at a single point not connected to a pad or via. These are potential stubs.

      Implementation: Build SpatialQueryEngine from pcb_primitives. For each trace path, query proximity to other paths on the same layer. If paths are within 2mm and run parallel for > 5mm, flag as crosstalk candidate.

      **PI Rick (Power Integrity):**
      - Category "decoupling": Find power nets (net name starts with "VCC", "VDD", "+3V3", "+5V", "GND"). Check if decoupling capacitors (footprints with "C" reference prefix on power nets) are within 10mm of IC power pins. Missing caps get a finding.
      - Category "power_plane_continuity": Find zones/copper pours on power nets and check for gaps (simplified: check if power zone coverage is > 50% of board area on power layers).

      Implementation: Use find_by_net() to get primitives on power nets. Use proximity queries to check cap-to-IC distances.

      **EMC Rick (Electromagnetic Compatibility):**
      - Category "trace_length": Find traces longer than 50mm. Flag as potential antenna. Severity based on length: > 50mm = "warning", > 100mm = "critical".
      - Category "clearance": Use spatial_violations (from enrich_drc_result) if provided. Filter for "clearance" type violations. Convert each to a RickFinding with EMC context.
      - Category "ground_plane_coverage": Check for ground zone/region primitives. If few or none, flag as concern.

      Implementation: Use find_by_net("GND") to find ground primitives. Calculate path lengths from SpatialPath.points.

      **DFM Rick (Design for Manufacturing):**
      - Category "component_density": Check if any 10mm x 10mm region contains > 20 footprints. Use containment queries on a sliding window.
      - Category "minimum_feature_size": Find traces narrower than 0.15mm (DFM constraint). Create findings for each.
      - Category "board_edge_clearance": Find primitives within 3mm of board edge (coordinates near (0,0), (width,0), (0,height), (width,height)).

      Implementation: Use containment() with sliding 10mm windows. Use find_by_layer() to get trace primitives, check width field.

      For each domain, if spatial_violations is provided and non-empty, also convert relevant violations to RickFindings (using violation coordinates and type).

      Build summary string: f"{len(findings)} findings: {critical_count} critical, {warning_count} warning, {info_count} info"

    - `generate_all_reports(pcb_primitives: list, spatial_violations: list[SpatialViolation] | None = None, board_path: str = "") -> dict[str, SpatialRickReport]`:
      Generate reports for all four Rick domains. Returns `{"si": SpatialRickReport, "pi": SpatialRickReport, "emc": SpatialRickReport, "dfm": SpatialRickReport}`.

    - `_calculate_path_length(points: tuple[tuple[float, float], ...]) -> float`:
      Sum of Euclidean distances between consecutive points in a SpatialPath.

    - `_classify_net_type(net_name: str) -> str`:
      Classify net as "power" (starts with VCC/VDD/+3V3/+5V/GND/VIN), "signal", or "unknown".
