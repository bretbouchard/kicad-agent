  <action>
    Create diff_pair.py and update __init__.py:

    1. diff_pair.py -- DiffPairResult frozen dataclass and route_differential_pair function:

       DiffPairResult fields:
       - net_positive: tuple of (x,y) waypoints
       - net_negative: tuple of (x,y) waypoints
       - length_positive_mm: float
       - length_negative_mm: float
       - mismatch_mm: float (absolute difference)
       - spacing_mm: float (target pair spacing)
       - valid: bool (True if both routes found and mismatch within tolerance)

       route_differential_pair parameters:
       - graph: RoutingGraph
       - source_pos: tuple[float, float] (positive net start)
       - source_neg: tuple[float, float] (negative net start)
       - target_pos: tuple[float, float] (positive net end)
       - target_neg: tuple[float, float] (negative net end)
       - net_name_pos: str = "DP_P"
       - net_name_neg: str = "DP_N"
       - target_spacing_mm: float = 0.15
       - max_length_mismatch_mm: float = 0.5

       Algorithm:
       a) Route positive net via route_net from Plan 01. If None, return invalid result.
       b) Route negative net via route_net. If None, return invalid result.
       c) Compute path lengths with _path_length helper.
       d) If mismatch > max_length_mismatch_mm, apply serpentining to shorter path:
          - _add_serpentining(path, target_delta_mm, grid_resolution_mm):
            Walk the path segments. For each straight segment long enough (>= 2 * amplitude),
            insert accordion bumps perpendicular to the segment direction.
            Each bump adds 2 * amplitude to the path length.
            Amplitude = min(target_spacing_mm * 2, remaining_delta / 4).
            Insert bumps at intervals along the segment until target_delta_mm is met.
            Bumps are a series of 4 points forming a U-shape: forward-perpendicular, back-perpendicular, forward.
          - Cap total serpentine additions to not exceed the target delta.
       e) Recompute lengths after serpentining.
       f) Return DiffPairResult with all computed fields.

       _path_length(path: list or tuple of (x,y)) -> float:
       Sum of Euclidean distances between consecutive points.

       _add_serpentining(path, delta_mm, grid_res, max_amplitude) -> tuple of (x,y):
       Insert accordion bumps into the shorter path to add delta_mm of length.
       Walk path segments, insert bumps perpendicular to segment direction.
       Bump geometry: from point A to point B along path, create detour:
         A -> A + perp*amplitude -> B_intermediate + perp*amplitude -> B_intermediate -> B
       Only insert where there is room (check that bump points are within board bounds).
