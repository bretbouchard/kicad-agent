       a. Define frozen dataclass LLMGenerationResult:
          - success: bool
          - intent: GenerationIntent | None -- parsed intent (stage 1 output)
          - generation_result: GenerationResult | None -- pipeline output (stage 2 output)
          - refinement_result: LLMRefinementResult | None -- refinement output (stage 3 output)
          - critique: CritiqueReport | None -- design critique (stage 4 output)
          - evaluation_result: EvaluationResult | None -- evaluation score (stage 5 output)
          - errors: tuple[str, ...] -- accumulated non-fatal errors from all stages

       b. Define llm_generate function:
          ```
          def llm_generate(
              description: str,
              output_dir: Path,
              run_refinement: bool = True,
              run_critique: bool = True,
              run_evaluation: bool = True,
              max_refinement_iterations: int = 5,
              intent_parser: IntentParser | None = None,
              design_critic: DesignCritic | None = None,
              error_fixer: ErrorFixer | None = None,
          ) -> LLMGenerationResult:
          ```
          Pipeline stages:

          Stage 1 - Intent Parsing:
          - Create IntentParser if not injected.
          - Call intent_parser.parse(description).
          - On ValueError or ValidationError: return LLMGenerationResult(success=False, errors=(str(e),)).
          - On success: continue with intent.

          Stage 2 - Design Generation:
          - Call generate_design(intent, output_dir, run_validation=True, run_export=False).
          - Export is deferred until after refinement (files may change).
          - If generation_result.success is False: return LLMGenerationResult(success=False, intent=intent, generation_result=generation_result, errors=generation_result.errors).
          - On success: continue.

          Stage 3 - LLM Refinement:
          - Only run if run_refinement=True AND generation_result.erc_pass is not True.
          - If schematic exists: call llm_refine_design(sch_path, pcb_path, max_iterations=max_refinement_iterations, error_fixer=error_fixer).
          - Store refinement_result.
          - If refinement not needed (ERC already passed): refinement_result = None.

          Stage 4 - Design Critique:
          - Only run if run_critique=True AND generation_result.pcb_path exists.
          - Parse PCB file via parse_pcb(pcb_path), build PcbIR, extract spatial primitives via extract_all().
          - Build SpatialQueryEngine from primitives.
          - Get ERC result for error context (re-run ERC if refinement ran).
          - Call design_critic.critique(engine, erc_result=erc_result).
          - If no PCB or critique disabled: critique = None.

          Stage 5 - Evaluation:
          - Only run if run_evaluation=True.
          - Call evaluate_design(generation_result, generation_result.project_dir).
          - Store evaluation_result.

          Final - Manufacturing Export:
          - If generation succeeded and PCB exists, attempt Gerber and BOM export (non-fatal failures).

          Build LLMGenerationResult:
          - success = generation succeeded AND (ERC passed OR refinement converged).
          - errors = accumulated errors from all stages (generation_result.errors + refinement errors + critique errors).

       c. The function signature accepts injected components (intent_parser, design_critic, error_fixer) for testability. When None, creates fresh instances.
