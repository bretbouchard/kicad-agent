    **L-1: Atomic file write in create_file.py**
    Add an `_atomic_write` helper function at the top of create_file.py:
    ```python
    import os
    import tempfile

    def _atomic_write(file_path: Path, content: str) -> None:
        """Write content atomically via temp file + rename to prevent TOCTOU races."""
        fd, tmp_path = tempfile.mkstemp(
            dir=file_path.parent,
            prefix=".kicad_",
            suffix=".tmp",
        )
        try:
            with os.fdopen(fd, 'w', encoding='utf-8') as f:
                f.write(content)
            os.rename(tmp_path, str(file_path))
        except BaseException:
            try:
                os.unlink(tmp_path)
            except OSError:
                pass
            raise
    ```
    Replace all direct `file_path.write_text(content, encoding='utf-8')` calls in create_file.py with `_atomic_write(file_path, content)`.
