Purpose: Provide the Python execution environment for all KiCad operations — without this, the app cannot edit .kicad_sch/.kicad_pcb files, run validation gates, or expose MCP tools.
Output: Code-signed PyInstaller daemon binary, ProcessManager for lifecycle control, DaemonMCPClient for stdio communication, audit-log trail for daemon events.
</objective>

Build the Python daemon with PyInstaller:
1. Sign all embedded Python dylibs (from /usr/local/opt/python@3.11/lib): `find /usr/local/opt/python@3.11/lib -name "*.dylib" -exec codesign --force --deep --sign "-" {} \;`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Swift App → Daemon | Trusted subprocess (local-only, no network) |
| Daemon → File System | Semi-trusted (KiCad file access, sandboxed) |
| Daemon Stdio | Trusted local IPC (no network exposure) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-162-01 | S | Stdio input (MCP requests) | mitigate | Validate JSON-RPC schema before daemon processing, sanitize file paths, reject absolute paths outside project dir. |
| T-162-02 | I | Daemon binary integrity | mitigate | SHA256 checksum verification on launch per APP-03, code signing with hardened runtime, dylib signing per Pitfall 1. |
| T-162-03 | D | Daemon file system access | mitigate | App Sandbox entitlements restrict daemon to user directories, validate file paths in MCP tool calls. |
| T-162-04 | E | Daemon crash/hang | mitigate | Watchdog timer (30s) kills hung daemon, crash loop detection halts after 5 crashes in 60s per DAEM-06. |
| T-162-05 | R | Daemon privileges (escalation) | accept | Daemon runs with same privileges as app (no special rights), no sudo/root required. |
| T-162-06 | T | Stdio eavesdropping | accept | Local-only IPC, no network exposure, pipes protected by macOS process isolation. |

## Adversarial Scenario

**Attack:** Malicious MCP request crafted to exploit daemon's stdio interface, causing buffer overflow or path traversal. **Mitigation:** (S) JSON-RPC schema validation rejects malformed requests, (D) Path sanitization rejects `../` escapes, (E) Watchdog kills hung daemon before corruption spreads, (I) Checksum verification ensures binary unmodified.
</threat_model>
