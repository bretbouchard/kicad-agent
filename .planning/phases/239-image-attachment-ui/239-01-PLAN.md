| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

<context>
## Why this exists
Gap A5 from GAP-ANALYSIS-CURRENT.md. ImageAttachmentView exists, KCAttachment exists, the model supports vision, but the UI doesn't expose attachment. The chat compose bar has no attachment affordance.

## Background
Phase 196 (UI automation) shipped some UI test plumbing but not the image-attach wiring. The router's buildKCPrompt explicitly says Image attachments from the chat UI are NOT yet bridged into KCAttachment here (RouterStreamProvider.swift:118-126).

## Scope (in)
- Attachment button in compose bar
- NSOpenPanel picker (PNG, JPEG, HEIC only)
- Drag-and-drop target on the compose bar
- Paste-from-clipboard handler (NSPasteboard.general)
- Thumbnail display in MessageBubbleView (already partial via ImageAttachmentView)
- Router integration: read bytes, attach to KCPrompt

## Scope (out)
- Multi-image per message UI (data model supports it; UI is single-image for v1)
- Camera capture (Phase 236 Vision Input)
- Vision-aware model selection (Gemma 4 12B V2 is the target; auto-routing comes later)
</context>

### Task 1: Image picker
Files: macos-app/Sources/KiCadAgent/Views/Chat/ChatView.swift

Add a paperclip button to the compose bar. Tapping it opens NSOpenPanel filtered to png/jpeg/heic. Selected file becomes an ImageAttachment in the pending message.

### Task 2: Drop target
Files: macos-app/Sources/KiCadAgent/Views/Chat/ChatView.swift

Add .onDrop(of: [.fileURL]) to the compose bar. Filter to image MIME types. Build an ImageAttachment from the dropped URL.

### Task 3: Paste handler
Files: macos-app/Sources/KiCadAgent/Views/Chat/ChatView.swift

Add NSEvent monitor for Cmd+V when compose bar is focused. If clipboard has image data, write to temp file and build ImageAttachment.

### Task 4: Validation + compression
Files: macos-app/Sources/KiCadAgent/Views/Chat/ChatView.swift

Run ImageAttachmentValidator. If needsCompression (over 10MB), resize to 2048px max and re-encode as JPEG. Strip EXIF.

### Task 5: Router integration
Files: macos-app/Sources/KiCadAgent/Views/Chat/RouterStreamProvider.swift

In buildKCPrompt, read each ImageAttachment.url as Data. Construct a KCAttachment for each. Pass to KCPrompt.attachments.

### Task 6: Tests
- Picker returns the selected file
- Drop delegate creates an ImageAttachment
- Paste from clipboard creates an ImageAttachment
- Compression resizes a 20MB image to under 10MB
- Router produces a KCPrompt with attachments
