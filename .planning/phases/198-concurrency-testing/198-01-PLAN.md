ThreadSanitizer detects races at runtime, catching bugs unit tests miss.

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-198-01 | T | False positive data race | mitigate | Review ThreadSanitizer report, verify race is real, fix code (no suppressions) |
| T-198-02 | R | CI workflow bypassed | mitigate | ThreadSanitizer required in main branch protection rules, bypass requires security review |
| T-198-03 | I | Suppressions file tampered | mitigate | Commit suppressions to git, PR review required for changes |
| T-198-04 | D | Test data corrupted by race | accept | Concurrency tests detect races, test data corruption is expected (test crashes) |
| T-198-05 | E | Third-party framework race | mitigate | Document suppression with third-party version, security review approves, reassess on upgrade |
