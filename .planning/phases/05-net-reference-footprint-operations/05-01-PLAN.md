```python
    """Add a new net to the PCB.

    Args:
        net_name: Net name. Empty string triggers auto-generation as "N_<number>".
        net_number: Explicit net number. None triggers auto-assignment (max existing + 1).

    Returns:
        The created Net object.

    Raises:
        ValueError: If net_name already exists (when explicitly named).
    """
```

Implementation: If net_name is empty, auto-generate "N_<number>". If net_number is None, find max(board.nets[].number) + 1. Create kiutils Net(number, name), append to self.board.nets. Call _record_mutation("add_net", {...}). Return the created Net.

```python
def remove_net(self, net_name: str) -> None:
    """Remove a net from the PCB, disconnecting all pads.

    Raises:
        ValueError: If net_name not found, or net_name is "" (net 0 is reserved).
    """
```

Implementation: Find Net by name in board.nets. For each footprint.pad, if pad.net and pad.net.name == net_name, set pad.net = None. Remove the Net from board.nets list. Call _record_mutation("remove_net", {...}).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| LLM intent -> Operation schema | Untrusted JSON input from LLM crosses into validated operation |
| Operation -> IR mutation | Validated operation triggers IR mutation on parsed file |
| IR -> kiutils object | Mutation methods modify kiutils Board/Schematic in memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | S (Spoofing) | AddNetOp net_name | mitigate | min_length=1, max_length=64 prevents empty/oversized names; TargetFile validation prevents path injection |
| T-05-02 | T (Tampering) | RemoveNetOp pad disconnect | mitigate | Only disconnects pads connected to the named net; ValueError on missing net prevents silent failures |
| T-05-03 | T (Tampering) | RenameNetOp propagation | mitigate | Propagation updates ALL connected pads atomically; ValueError if new_name already exists prevents collision |
| T-05-04 | D (Denial) | AddNetOp net_number exhaustion | accept | KiCad net numbers are 16-bit; practical limit far exceeds reasonable use |
| T-05-05 | I (Info Disclosure) | Bus member_nets list | accept | Bus members are design metadata, not sensitive information |
| T-05-06 | E (Elevation) | RemoveBusOp member cleanup | mitigate | Cleanup only removes bus entry references, not arbitrary schematic objects |
</threat_model>
