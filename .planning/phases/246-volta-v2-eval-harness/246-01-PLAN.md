## Hardware Contingency Plan

Default: NVIDIA A100 40GB or better. 4-bit quantization fits in <24GB VRAM with batch=1, max_new_tokens=512.

Fallback paths:
| Failure | Response |
|---|---|
| GPU OOM mid-case | Catch OOM, retry with empty_cache + halve max_new_tokens to 256. If still OOM, mark case as `error_class: model_oom` and continue |
| No GPU at all (CPU-only host) | Auto-detect CUDA unavailable, switch to `device="cpu"` + `quantization="none"` (full FP16) |
| GPU present but <16GB VRAM | Detect via `torch.cuda.get_device_properties(0).total_memory`. If <16GB, refuse to start unless `--allow-low-vram` flag is passed |
| bitsandbytes unavailable | Fall back to FP16 on GPU, requires ~24GB VRAM. Refuse if less |
| Vast.ai relay drops mid-download | HF download retries 3x with exponential backoff. If all fail, exit with code 2 |

## Task 0 (prerequisite): Verify Volta v2 adapter availability on HF

Create `tests/eval/verify_hf_availability.py`:
```python
"""Verify Volta v2 adapter is published on HuggingFace.

Run as a prerequisite to the eval harness. Exits 0 if adapter is
available with the required files, 2 if not.
"""
import sys
import requests

ADAPTER_REPO = "bretbouchard/volta-pcb-adapter-v2"
REQUIRED_FILES = [
    "adapter_config.json",
    "adapter_model.safetensors",
    "tokenizer.json",
    "tokenizer_config.json",
    "chat_template.jinja",
]
EXPECTED_SAFETENSORS_SIZE = 524649216  # 524MB

def main():
    r = requests.get(f"https://huggingface.co/api/models/{ADAPTER_REPO}", timeout=10)
    if r.status_code == 404:
        print(f"FAIL: {ADAPTER_REPO} not found on HuggingFace")
        return 2
    r.raise_for_status()
    data = r.json()
    files = {s["rfilename"] for s in data.get("siblings", [])}
    missing = [f for f in REQUIRED_FILES if f not in files]
    if missing:
        print(f"FAIL: missing files in {ADAPTER_REPO}: {missing}")
        return 2
    print(f"OK: {ADAPTER_REPO} has all {len(REQUIRED_FILES)} required files")
    print(f"  private: {data.get('private')}, gated: {data.get('gated')}")
    return 0

if __name__ == "__main__":
    sys.exit(main())
```

Run `python tests/eval/verify_hf_availability.py` before Task 1.
