      1. For each (ref, (x, y, rotation)) in positions:

         - Create SpatialBox with computed corners
      2. Return list of SpatialBox

        1. Build boxes from positions + sizes using positions_to_boxes()
        2. Check pairwise clearances using SpatialQueryEngine:
           a. Build SpatialQueryEngine from all boxes
           b. For each box, query clearance() within search_radius = 2 * min_clearance
           c. Report violations where distance < min_clearance
        3. Check board bounds: each box must be within (margin, margin, board_width - margin, board_height - margin)
        4. Check overlap: any pair with distance == 0 is an overlap (critical)
        5. Return (is_valid, violations_list)
        6. Validate: raise ValueError if board_width <= 0 or component count > 500

            "board_utilization": float,  # sum of component areas / board area

    **1. Create src/kicad_agent/placement/scoring.py:**

    - `PlacementScore(frozen=True)`: Dataclass with:
      - total_score: float  # Weighted combination [0, 1]
      - hpwl: float  # Half-perimeter wirelength (mm)
      - hpwl_normalized: float  # HPWL normalized by board diagonal * component count [0, 1]
      - congestion_estimate: float  # Estimated routing congestion [0, 1], lower is better
      - clearance_score: float  # Fraction of component pairs with adequate clearance [0, 1]
      - edge_score: float  # Penalty for components near board edges [0, 1]
      - board_utilization: float  # Total component area / board area

    - `compute_hpwl_score(positions: dict[str, tuple[float, float, float]], graph: PlacementGraph) -> tuple[float, float]`:
      Compute HPWL and normalized HPWL.
      1. For each net in the graph (net nodes):
         - Find all component nodes connected to this net
         - Get (x, y) positions for those components
         - Compute bounding box: min_x, max_x, min_y, max_y
         - HPWL contribution = (max_x - min_x) + (max_y - min_y)
      2. Sum all net contributions -> total HPWL
      3. Normalize: hpwl_normalized = 1.0 - (hpwl / (board_diagonal * n_components))
      4. Return (hpwl, max(0.0, hpwl_normalized))

    - `compute_congestion_estimate(positions: dict[str, tuple[float, float, float]], graph: PlacementGraph, grid_resolution: int = 10) -> float`:
      Estimate routing congestion using a grid-based approach.
      1. Overlay a grid_resolution x grid_resolution grid on the board
      2. For each net, draw a bounding box between connected components
      3. Increment congestion counter for each grid cell intersected by bounding boxes
      4. Congestion estimate = max_cell_congestion / average_cell_congestion (or 0 if no nets)
      5. Cap at 1.0. Lower is better (more uniform routing density).

    - `PlacementScorer`:
      - `__init__(board_width: float, board_height: float, min_clearance: float = 1.0)`:
        Store board dimensions and clearance rule.

      - `score(positions: dict[str, tuple[float, float, float]], graph: PlacementGraph, component_sizes: dict[str, float] | None = None) -> PlacementScore`:
        1. Compute HPWL: call compute_hpwl_score()
        2. Compute congestion: call compute_congestion_estimate()
        3. Compute clearance score: build boxes, use PlacementValidator.validate(), compute fraction of pairs with adequate clearance
        4. Compute edge score: penalty for components within 2*min_clearance of board edges
        5. Compute board utilization: sum of component areas / board area
        6. Compute total_score = 0.3 * hpwl_normalized + 0.2 * (1.0 - congestion) + 0.3 * clearance_score + 0.2 * edge_score
        7. Return PlacementScore
