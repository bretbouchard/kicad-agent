<objective>
Connect EasyEDA format support with the LCSC component library and JLCPCB manufacturing pipeline. This plan builds the integration layer that makes EasyEDA support commercially useful: search LCSC for parts, map between LCSC and KiCad, and export BOMs ready for JLCPCB assembly.

Purpose: EasyEDA's primary value proposition is JLCPCB assembly integration. Without LCSC/JLCPCB support, we have format conversion but not the workflow that EasyEDA users actually need. This plan bridges that gap.

Output: LCSC component mapper, JLCPCB BOM exporter, and EasyEdaClient extracted from the existing MCP server.
</objective>

       ```python
       }

       # Reverse index: (symbol, footprint) -> LCSC part number
       _KICAD_TO_LCSC: dict[tuple[str, str], str] = {
           v: k for k, v in _LCSC_TO_KICAD.items()
       }


       class LcscComponentMapper:
           """Bidirectional LCSC <-> KiCad component mapping.

           Maps LCSC part numbers to KiCad symbol library IDs and footprint
           assignments, and vice versa. Covers top 50 most common components
           for JLCPCB assembly.

           Usage:
               mapper = LcscComponentMapper()
               symbol, footprint = mapper.lcsc_to_kicad("C25804")
               lcsc = mapper.kicad_to_lcsc("Device:R", "Resistor_SMD:R_0805_2012Metric")
           """

           def lcsc_to_kicad(self, lcsc_part: str) -> Optional[tuple[str, str]]:
               """Map LCSC part number to (KiCad symbol, KiCad footprint).

               Args:
                   lcsc_part: LCSC part number (e.g., "C25804").

               Returns:
                   Tuple of (symbol_lib_id, footprint) or None if unknown.
               """
               return _LCSC_TO_KICAD.get(lcsc_part.upper())

           def kicad_to_lcsc(self, symbol: str, footprint: str) -> Optional[str]:
               """Map KiCad symbol + footprint to LCSC part number.

               Args:
                   symbol: KiCad symbol library ID (e.g., "Device:R").
                   footprint: KiCad footprint (e.g., "Resistor_SMD:R_0805_2012Metric").

               Returns:
                   LCSC part number or None if unknown.
               """
               return _KICAD_TO_LCSC.get((symbol, footprint))

           def entry_count(self) -> int:
               """Return number of entries in the mapping table."""
               return len(_LCSC_TO_KICAD)
       ```
