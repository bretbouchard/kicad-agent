    **Step 1: Create `constraints/coordinate.py` per D-50-05.**
    Define `CoordinateConverter` as a frozen dataclass:
    ```python
    @dataclass(frozen=True)
    class CoordinateConverter:
        board_height_mm: float
        offset_x: float = 0.0
        offset_y: float = 0.0
        rotation_deg: float = 0.0
        scale: float = 1.0
    ```

    Implement `schematic_to_pcb(pos: tuple[float, float]) -> tuple[float, float]`:
    1. Apply Y-flip: `(x, board_height_mm - y)`
    2. Apply scale: multiply both coords by self.scale
    3. Apply rotation (if rotation_deg != 0): standard 2D rotation matrix
    4. Apply offset: add self.offset_x, self.offset_y

    Implement `pcb_to_schematic(pos: tuple[float, float]) -> tuple[float, float]`:
    Inverse of the above: subtract offset, inverse rotation, divide by scale, inverse Y-flip.
    Use `math.cos`, `math.sin`, `math.radians` for rotation. Precompute `cos_r` and `sin_r` from `rotation_deg`.

    **Step 2: Create `constraints/converters.py` per D-50-03.**
    These are pure functions -- no side effects, no class state.

    `to_routing_constraints(constraints: list[PCBConstraint]) -> RoutingConstraints`:
    - Extract ClearanceConstraint instances -> use min_clearance_mm for clearance_mm
    - Extract ImpedanceConstraint instances -> use trace_width_mm for trace_width_mm
