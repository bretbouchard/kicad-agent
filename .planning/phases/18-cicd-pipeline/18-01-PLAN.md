  <name>Task 2: Create GitHub Actions CI workflow</name>
  <files>.github/workflows/ci.yml</files>
  <action>
Create `.github/workflows/ci.yml` with three parallel jobs: test, lint, typecheck.

Workflow trigger:
- `on.push.branches: [master]`
- `on.pull_request.branches: [master]`

**test job:**
- `runs-on: ubuntu-latest`
- Strategy matrix: `python-version: ["3.11"]`
- Steps:
  1. `actions/checkout@v6` with `fetch-depth: 0` (required by setuptools-scm per RESEARCH Pitfall 1)
  2. `actions/setup-python@v6` with `python-version: ${{ matrix.python-version }}` and `cache: pip`
  3. `run: pip install -e ".[dev]"`
  4. `run: python -m pytest tests/ -x -v --tb=short --cov=kicad_agent` -- do NOT pass --cov-fail-under here; the fail_under in pyproject.toml [tool.coverage.report] is the single source of truth

**lint job:**
- `runs-on: ubuntu-latest`
- Steps:
  1. `actions/checkout@v6`
  2. `actions/setup-python@v6` with `python-version: "3.11"` and `cache: pip`
  3. `run: pip install ruff`
  4. `run: ruff check src/ tests/`

**typecheck job:**
- `runs-on: ubuntu-latest`
- Steps:
  1. `actions/checkout@v6`
  2. `actions/setup-python@v6` with `python-version: "3.11"` and `cache: pip`
  3. `run: pip install -e ".[dev]"`
  4. `run: python -m mypy src/kicad_agent`

Create the `.github/workflows/` directory structure. Use YAML with 2-space indentation.
  </action>
