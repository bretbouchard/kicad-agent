SpiceLib AscEditor API (verified spicelib 1.5.1):
```python
from spicelib import AscEditor
AscEditor.set_custom_library_paths(str(stubs_dir))  # CLASS method, affects all instances
editor = AscEditor("circuit.asc")
editor.get_components()          # -> list[str] of references ["R1", "C1", "V1"]
editor.get_component("R1")       # -> SchematicComponent with .position, .rotation, .reference
editor.get_component_position("R1")  # -> tuple[Point, ERotation]
editor.get_component_value("R1")     # -> str "1k"
editor.get_component_info("R1")      # -> dict {'Value': '1k', 'InstName': 'R1'}
editor.wires          # -> list[Line] where Line.V1, Line.V2 are Points, Line.net is str
editor.labels         # -> list[Text] where Text.coord is Point, Text.text is net name
editor.directives     # -> list[Text] where Text.text is command, Text.type is TextTypeEnum
```

SpiceLib structures:
```python
class Point:
    X: float
    Y: float

class Line:
    V1: Point
    V2: Point
    net: str
    style: LineStyle
    def touches(point: Point) -> bool

class Text:
    coord: Point
    text: str
    size: int
    type: TextTypeEnum
    angle: ERotation

class ERotation(IntEnum):
    R0=0, R45=45, R90=90, R135=135, R180=180, R225=225, R270=270, R315=315,
    M0=360, M45=405, M90=450, M135=495, M180=540, M225=585, M270=630

class TextTypeEnum(IntEnum):
    NULL, COMMENT, DIRECTIVE, LABEL, ATTRIBUTE, PIN

class SchematicComponent:
    position: Point
    rotation: ERotation
    reference: str
    attributes: OrderedDict
    symbol: object  # AsyReader or None
```
