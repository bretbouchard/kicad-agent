<tasks>

### Task 1: Install fastlane + Ruby env
Files: Gemfile (new), .ruby-version (new)

Pin Ruby 3.2+ in .ruby-version. Add Gemfile with fastlane, fastlane-plugin-versioning, xcodeproject. Run bundle install.

### Task 2: Match repo
Set up a private git repo (e.g., github.com/bretbouchard/kicad-agent-certs) for match storage. Configure Matchfile to point at it. Initial match run generates the Developer ID Application + Developer ID Installer certs.

### Task 3: Appfile + Matchfile
Files: fastlane/Appfile, fastlane/Matchfile

Appfile reads APPLE_ID, TEAM_ID, ITC_TEAM_ID from env. Matchfile points at the match repo.

### Task 4: Fastfile
Files: fastlane/Fastfile

Lanes:
- mac build — debug build, no signing
- mac build_release — release build with match appstore signing + notarization
- mac beta — build_release + pilot upload to TestFlight
- mac release — build_release + deliver to App Store
- mac snapshot — run snapshot to generate App Store screenshots
- mac build_daemon — PyInstaller daemon build + codesign --deep --sign
- mac ci — full pipeline (build, test, build_daemon, build_release, beta if on main)

### Task 5: PyInstaller lane
Files: fastlane/Fastfile (build_daemon lane), Scripts/resign_kicad_daemon.sh

PyInstaller builds the daemon binary; shell script applies hardened runtime + Developer ID signature.

### Task 6: Snapshot setup
Files: fastlane/Snapfile, fastlane/screenshots/

Configure snapshot to capture the 3 starter projects from the onboarding tour. Run once to populate the screenshot directory.

### Task 7: Deliver metadata
Files: fastlane/metadata/en-US/

Pull from docs/APP-STORE-DESCRIPTION.md, docs/ENGINEERING-DESCRIPTION.md. Add keywords, release notes, support URL, marketing URL.

### Task 8: CI wiring
Files: .github/workflows/release.yml (or scripts/ci.sh)

Trigger on tag push. Runs mac ci lane. Uploads the .app.zip + .dmg as release artifacts.

### Task 9: Verification
- dry-run match: certs sync
- dry-run fastlane mac build: produces a .app
- dry-run fastlane mac snapshot: produces screenshots
- Actual fastlane mac beta: requires real Apple ID + match repo; document but don't run in CI
</tasks>
