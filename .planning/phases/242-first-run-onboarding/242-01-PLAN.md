Gap F3 from GAP-ANALYSIS-CURRENT.md. The app opens to a blank project sidebar with no guidance. First-time users don't know what to do.

KiCadInstallView was the previous onboarding, but Phase 220 removed the install path. The view is orphaned on disk.

## Scope (in)
- 3-step guided tour:
  1. Welcome + pick a starter project (LED blinker, ESP32 breakout, op-amp preamp)
  2. Run a chat (e.g., "Make the LED blink at 1 Hz")
  3. View the result + Open in KiCad button
- Skip / dismiss always available
- Persisted state in SwiftData (dismissed: Bool, completed: Bool)
- Returning user with an open project skips the tour
- Returning user with no projects gets a re-entry button in the sidebar

- Model setup wizard (BYOK flow handles that in settings)
