## Adversarial Scenario

A malicious .kicad_sch file in the corpus contains deeply nested XML structures designed to cause parsing stack overflow in the Swift SchematicParser, crashing the batch process and blocking the entire parity test. Mitigation: CircuitBreaker pattern in both Python (try/except with per-schematic timeout) and Swift (parsing with depth limits); log error and skip to next schematic on parse failure.</threat_model>
