## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Model output → Python dict | Untrusted free-text from a vision model crosses into structured data. Parser must never raise; always return {} on failure. |
| Python dict → RoutingStrategyResult | Model-emitted values (net names, layers, backends) cross into the Phase 100 contract. Translator must apply safe defaults (ASTAR) for any unrecognized value. |
| Orchestrator → AiRoutingStrategy | The orchestrator trusts RoutingStrategyResult from DeterministicStrategy. Phase 98 output is untrusted — the H4 gate in orchestrator.py:127-151 is the structural backstop. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-98-01-01 | Tampering | parse_strategy_json | mitigate | Parser returns {} on any malformed input; never raises. Caller treats {} as "trigger fallback." |
| T-98-01-02 | Elevation of Privilege | _translate_to_result | mitigate | Unknown backend strings default to RouterBackend.ASTAR (lowest-privilege backend — cannot dispatch to Freerouting without explicit model consent). |
| T-98-01-03 | Information Disclosure | build_strategy_prompt | accept | Prompt surfaces net names to the model. These are already in the .kicad_pcb file the model is analyzing — no new disclosure. |
| T-98-01-04 | Denial of Service | AiRoutingStrategy.strategize | mitigate | _AiStrategyError raised on empty output / render failure. Plan 02 wraps this in fallback. No infinite loops. |
| T-98-01-05 | Repudiation | AiRoutingStrategy | accept | Audit trail is the orchestrator's responsibility (Phase 100). Plan 01 does not touch audit. |
