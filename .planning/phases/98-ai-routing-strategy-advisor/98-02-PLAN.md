    ```python
    class StrategyValidator:
        def __init__(self, board=None):
            # board: NativeBoard (optional — needed for layer stackup validation).
            # If None, layer validation uses {F.Cu, B.Cu} default.

        def validate(
            self,
            result: RoutingStrategyResult,
            board_state: BoardState,
            netlist: dict[str, list[Pin]],
        ) -> None:
            """Raise ValueError on any invalid field. Returns None on success."""
            self._validate_keepouts(result.keepouts, board_state.board_bounds)
            self._validate_net_references(result, netlist)
            self._validate_layer_hints(result.layer_hints)
            self._validate_keepout_layers(result.keepouts)
    ```

    Each sub-validator raises ValueError with a descriptive message including the offending value (per RESEARCH.md pseudocode in Validation Gate Architecture).

    `_validate_keepouts(keepouts, board_bounds)`:
    - Unpack `min_x, min_y, max_x, max_y = board_bounds`.
    - For each keepout:
      - Check `min_x <= keepout.x1 <= max_x` and same for x2, y1, y2. Raise with message on violation.
      - Check `keepout.x1 < keepout.x2` and `keepout.y1 < keepout.y2`. Raise "zero/negative area" on violation.

    `_validate_net_references(result, netlist)`:
    - `known = set(netlist.keys())`.
    - For net in result.net_priorities: raise if not in known.
    - For net in result.router_assignment: raise if not in known.
    - For net in result.layer_hints: raise if not in known.
    - `missing = known - set(result.router_assignment.keys())`. Raise if missing (every net needs an assignment).
    - `missing_prio = known - set(result.net_priorities)`. Raise if missing (priority list must cover all nets).

    `_validate_layer_hints(layer_hints)`:

    - For net, layer in layer_hints.items(): raise if layer not in valid_layers.

    ```python
        try:
            image = self._render_fn(self._pcb_path)

            raw = self._pipeline.generate_from_image(image, prompt)
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| AiRoutingStrategy → StrategyValidator | Model-emitted RoutingStrategyResult crosses into validated territory. Validator is the last semantic check before the orchestrator consumes it. |
| StrategyValidator → DeterministicStrategy | On validation failure, control transfers to trusted deterministic code. This boundary is one-way — no path back to AI. |
| Orchestrator H4 → AiRoutingStrategy output | Phase 100's structural validation runs AFTER R-4. If R-4 passes but H4 catches something R-4 missed, the orchestrator raises ValueError — but this is Phase 100's contract, not Plan 02's. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-98-02-01 | Tampering | StrategyValidator._validate_keepouts | mitigate | Reject any coordinate outside board_bounds. Model cannot inject keepouts that route around security-critical regions. |
| T-98-02-02 | Tampering | StrategyValidator._validate_layer_hints | mitigate | Reject In3.Cu on 2-layer boards. Prevents model from "assigning" nets to nonexistent layers that could confuse downstream tools. |
| T-98-02-03 | Spoofing | StrategyValidator._validate_net_references | mitigate | Reject unknown net names. Model cannot introduce phantom nets that don't exist in the netlist. |
| T-98-02-04 | Denial of Service | AiRoutingStrategy.strategize fallback | mitigate | Broad except ensures any model failure → deterministic fallback. No crash path reaches the orchestrator. |
| T-98-02-05 | Elevation of Privilege | Fallback routing_notes | accept | routing_notes records the failure reason. This is informational, not privileged — acceptable disclosure for audit trail. |
| T-98-02-06 | Repudiation | Fallback path logging | mitigate | logger.warning records every fallback with the exception type and message. Audit trail (Phase 100) captures routing_notes with "ai_fallback:" prefix. |
