Build the parser layer that reads all four KiCad file types into kiutils Python objects, with raw content preservation for downstream UUID extraction and a sexpdata fallback parser.

Purpose: This is the foundation -- every subsequent phase depends on being able to parse KiCad files into structured Python objects. The parsers must preserve raw file content for UUID extraction (kiutils drops PCB/footprint UUIDs) and support sexpdata fallback for unknown constructs.
