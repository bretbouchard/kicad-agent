1. **CoordinateTransformer** -- frozen dataclass with configurable scale and sheet_height:
```python
@dataclass(frozen=True)
class CoordinateTransformer:
    """Convert KiCad mm coordinates to LTspice internal units.

    KiCad: mm, Y increases downward (screen coords).
    LTspice: internal units, Y increases upward (math coords).
    Default scale: 16 LTspice units per mm.
    Grid alignment: all outputs rounded to multiples of 16.
    """
    scale: int = 16
    sheet_height_mm: float = 297.0  # A4 height

    def mm_to_ltspice(self, kicad_x_mm: float, kicad_y_mm: float) -> tuple[int, int]:
        raw_x = kicad_x_mm * self.scale
        raw_y = (self.sheet_height_mm - kicad_y_mm) * self.scale
        ltspice_x = int(round(raw_x / 16)) * 16
        ltspice_y = int(round(raw_y / 16)) * 16
        return (ltspice_x, ltspice_y)
```

2. **AscWriter** class that:
   - Takes a kiutils Schematic, SymbolMapper, and CoordinateTransformer in constructor
   - Has method `write(output_path: str | Path) -> Path` that:
     a. Creates a minimal .asc template in a temp file
     b. Bootstraps an AscEditor from that template
     c. Iterates schematic.schematicSymbols:
        - Calls symbol_mapper.map_symbol(lib_id) for each
        - For COMPONENT type: creates SchematicComponent with reference, symbol, position (via coordinate transformer), rotation (maps KiCad angle to ERotation via ASC_ROTATION_DICT), value
        - For FLAG type: creates a FLAG entry (Text with LABEL type) at the symbol's position
        - For UNMAPPED: logs a warning, skips the component
     d. Iterates graphicalItems to find Connection items (wire segments):
        - Extracts (x1,y1,x2,y2) from Connection points
        - Transforms each endpoint via coordinate transformer
        - Appends Line objects to editor.wires
     e. Iterates labels and globalLabels:
        - Strips leading "/" from label text
        - Decodes "{slash}" to "/" in label text
        - Creates FLAG entries at label positions
     f. Calls editor.save_netlist(output_path)
     g. Cleans up temp file
     h. Returns the output Path

   Helper method `_rotation_from_kicad(angle, mirror)`:
     - Maps KiCad angle (0/90/180/270) + mirror to ERotation
     - Uses from spicelib.editor.asc_editor import ASC_ROTATION_DICT
     - Mirror maps to M0/M90/M180/M270 variants

   Helper method `_sanitize_net_name(text: str) -> str`:
     - Strip leading "/"
     - Decode "{slash}" to "/"
     - Strip whitespace

3. **Top-level convenience function:**
```python
def export_schematic_to_asc(
    kicad_sch_path: str | Path,
    output_path: str | Path,
    custom_mappings: dict[str, str] | None = None,
) -> Path:
    """Export a KiCad schematic to LTspice .asc format.

    Args:
        kicad_sch_path: Path to .kicad_sch file.
        output_path: Path for output .asc file.
        custom_mappings: Optional custom symbol mappings.

    Returns:
        Path to the written .asc file.
    """
    from kiutils.schematic import Schematic
    schematic = Schematic.from_file(str(kicad_sch_path))
    mapper = SymbolMapper(custom_mappings=custom_mappings)
    transformer = CoordinateTransformer()
    writer = AscWriter(schematic, mapper, transformer)
    return writer.write(output_path)
```

IMPORTANT implementation notes from research:
- Do NOT use editor.add_instruction() for directives -- it has broken behavior (replaces instead of appends). Use direct list manipulation (editor.wires.append, editor.labels.append, editor.directives.append).
- Use `from spicelib.editor.asc_editor import ASC_ROTATION_DICT` to map "R0" string to ERotation.R0 enum.
- Use tempfile.NamedTemporaryFile for the template, clean up in a finally block.
- Import kiutils Schematic lazily in export_schematic_to_asc only (not at module level) to match the project pattern of lazy imports for optional dependencies.
