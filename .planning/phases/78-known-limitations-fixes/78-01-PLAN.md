Add held-out evaluation for the reward model using ground-truth quality scores from score_chain(), computing Kendall tau correlation and top-k accuracy metrics. This decouples reward model quality assessment from the circular self-scoring pattern in evaluation.py (per D-M14, M-14).

Purpose: The reward model currently tracks val_loss (MSE) during training but never evaluates whether its ranking of chains correlates with actual quality. The discrimination_gap metric in evaluation.py uses the reward model to score chains -- circular evaluation. This plan adds a proper held-out eval function using rule-based ground truth.

Output: eval_reward_quality() function in reward_model.py, re-exported from evaluation.py, with tests.

    ```python
    def eval_reward_quality(
        model_predictions: list[PredictedReward],
        ground_truth_labels: list[tuple[float, float, float]],
    ) -> dict[str, float]:
        """Evaluate reward model quality against ground-truth scores.

        Computes rank correlation metrics between model predictions and
        ground-truth quality labels. Uses rule-based ground truth (from
        score_chain()) to avoid circular evaluation where the model scores
        its own outputs.

        Args:
            model_predictions: List of PredictedReward from the model.
            ground_truth_labels: List of (format, quality, accuracy) tuples
                from rule-based scoring. Same length as model_predictions.

        Returns:
            Dict with keys:
                kendall_tau: Kendall rank correlation coefficient.
                spearman_rho: Spearman rank correlation coefficient.
                top_1_accuracy: Fraction where model's #1 matches truth's #1.
                top_3_accuracy: Fraction where truth's #1 is in model's top-3.
                n_samples: Number of samples evaluated.

        Raises:
            ValueError: If input lists have different lengths or are empty.
        """
    ```

    Implementation steps:
    1. Compute composite scores: model_avg = (pred.format_score + pred.quality_score + pred.accuracy_score) / 3.0 and truth_avg = (gt[0] + gt[1] + gt[2]) / 3.0

    3. Use scipy.stats.spearmanr(model_avgs, truth_avgs) for Spearman rho
    4. Sort indices by model scores descending, check if index of max truth_avg is in top-1 and top-3
    5. Return dict with all five metrics

    Also add lazy import guard: wrap scipy import in try/except ImportError with numpy fallback (numpy.argsort for rank correlation).
