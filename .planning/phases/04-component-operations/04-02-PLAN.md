    1. Create `src/kicad_agent/ops/array_replicate.py`:
       - `ArrayReplicateError` exception class
       - `_linear_positions(source: Position, count: int, spacing: PositionSpec) -> list[Position]`:
         a. Generate positions: source + spacing * i for i in range(1, count+1)
         b. Each position preserves source angle
       - `_circular_positions(source: Position, count: int, center: PositionSpec, angle_step: float) -> list[Position]`:
         a. For each step i in range(1, count+1):
            - Calculate angle = source_angle + angle_step * i (in degrees)
            - Convert to radians for cos/sin
            - Calculate dx = source.X - center.x, dy = source.Y - center.y
            - Rotate (dx, dy) by angle_step * i
            - New position: (center.x + rotated_dx, center.y + rotated_dy, angle)
         b. Return list of Position objects
       - `_matrix_positions(source: Position, rows: int, cols: int, spacing: PositionSpec) -> list[Position]`:
         a. For row in range(rows), col in range(cols), skipping (0,0) which is the source:
            - position = (source.X + col * spacing.x, source.Y + row * spacing.y)
         b. Return list of Position objects
       - `array_replicate(op: ArrayReplicateOp, ir: SchematicIR) -> dict`:
         a. Find source component via ir.get_component_by_ref(op.source_reference)
         b. If not found, raise ArrayReplicateError
         c. Validate pattern-specific parameters:
            - linear: requires count and spacing
            - circular: requires count, center, and angle_step
            - matrix: requires rows, cols, and spacing
         d. Generate positions based on pattern type
         e. For each position, deep copy the source symbol (reuse _deep_copy_symbol pattern from duplicate):
            - Fresh UUID for symbol and all pins
            - Incremented reference
            - New position
            - Append to ir.components
         f. Record mutation with pattern type and count
         g. Return dict with created references, UUIDs, and positions

| Geometric calculation -> Position | Computed positions are applied to kiutils Position objects |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07 | D (Denial of service) | array_replicate count | mitigate | Pydantic Field constraint: count <= 100 (prevent creating thousands of components) |
| T-04-08 | T (Tampering) | duplicate reference increment | mitigate | _increment_reference scans all existing references -- no collision possible |
| T-04-09 | T (Tampering) | circular array position | accept | Floating-point precision in cos/sin is acceptable for schematic placement |
| T-04-10 | S (Spoofing) | deep_copy_symbol UUIDs | mitigate | Generate fresh UUIDs via uuid.uuid4() -- never copy source UUIDs |
</threat_model>
