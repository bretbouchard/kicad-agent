<objective>
Library reference propagation -- when a symbol or footprint library reference changes,
all instances across schematic and PCB files update to match.

Purpose: When a user renames a symbol in a library or moves a footprint to a different
library, every schematic component and PCB footprint referencing the old name must be
updated to the new name. Without propagation, broken references cause ERC/DRC errors.

Output: propagate_symbol_ref and propagate_footprint_ref functions with PropagationResult.
</objective>
