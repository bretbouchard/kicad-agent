<objective>
Structural diff generation for KiCad S-expression files with difftastic integration.

Purpose: When comparing two versions of a KiCad file, a raw text diff is noisy because
S-expressions have deep nesting and positional semantics. A structural diff understands
the S-expression hierarchy and reports differences at the element level (e.g., "component
R1 changed value from 10k to 4.7k" rather than a line-level diff of the surrounding
parentheses).

Output: structural_diff function, DiffEntry/DiffResult types, difftastic integration, test suite.
</objective>

  <behavior>
    - DiffType enum: ADDED, REMOVED, MODIFIED, MOVED
    - DiffEntry dataclass: diff_type, element_type (symbol, wire, footprint, etc.), identifier, old_value, new_value, path_in_file
    - DiffResult dataclass: entries, file_a_path, file_b_path, difftastic_available
    - structural_diff(file_a, file_b) parses both files and produces DiffResult
    - _extract_elements(sexp) groups top-level S-expressions by their first atom (element type)
    - _diff_element_groups(groups_a, groups_b) compares element groups and produces DiffEntry list
    - _try_difftastic(file_a, file_b) calls difftastic subprocess if available, returns Optional[str]
    - Identifier extraction: uses uuid or reference designator as element identifier
  </behavior>
  <action>
    1. Create `src/kicad_agent/crossfile/diff.py` with:

       - `DiffType` enum (inherit from str, Enum):
         - `ADDED = "added"` -- element exists only in file_b
         - `REMOVED = "removed"` -- element exists only in file_a
         - `MODIFIED = "modified"` -- element exists in both but content differs
         - `MOVED = "moved"` -- element position changed (detected by "at" field change)

       - `DiffEntry` frozen dataclass:
         - `diff_type: DiffType`
         - `element_type: str` -- e.g., "symbol", "wire", "footprint", "net"
         - `identifier: str` -- uuid or reference that identifies this element
         - `old_value: Optional[str]` -- serialized form in file_a (None for ADDED)
         - `new_value: Optional[str]` -- serialized form in file_b (None for REMOVED)
         - `path_in_file: str` -- structural path like "/symbol/uuid-abc"

       - `DiffResult` frozen dataclass:
         - `entries: list[DiffEntry]`
         - `file_a_path: Path`
         - `file_b_path: Path`
         - `difftastic_available: bool`
         - `difftastic_output: Optional[str]` -- raw difftastic text if available

       - `structural_diff(file_a: Path, file_b: Path) -> DiffResult`:
         - Read both files, parse with `parse_raw_sexp` from raw_parser
         - Call `_extract_elements` on both to get grouped elements
         - Call `_diff_element_groups` to compare and produce DiffEntry list
         - Call `_try_difftastic` for enhanced text diff
         - Return DiffResult

       - `_extract_elements(sexp: list) -> dict[str, dict[str, str]]`:
         - Input: parsed S-expression (nested list from parse_raw_sexp)
         - Find the top-level container (first element is "kicad_sch" or "kicad_pcb")
         - Iterate its children, group by their first atom (element type)
         - For each element, extract an identifier:
           - Look for `(uuid "...")` child -- use that as identifier
           - Fall back to `(property "Reference" "...")` for symbols
           - Fall back to index-based identifier
         - Return dict: `{element_type: {identifier: serialized_form}}`
         - Serialized form = re-serialize the S-expression sub-list back to string
           using a simple S-expression serializer (walk the nested list, emit parens and atoms)

       - `_diff_element_groups(groups_a, groups_b) -> list[DiffEntry]`:
         - For each element type present in either group:
           - Find identifiers only in A -> REMOVED entries
           - Find identifiers only in B -> ADDED entries
           - Find identifiers in both -> compare serialized forms:
             - If content differs -> MODIFIED
             - Optionally detect position changes -> MOVED (check "at" field)
         - Return sorted list of DiffEntry

       - `_sexp_to_string(sexp) -> str`:
         - Simple recursive S-expression serializer: atoms become strings,
           lists become "(item1 item2 ...)" with proper spacing
         - Handles nested lists and quoted strings

       - `_try_difftastic(file_a: Path, file_b: Path) -> tuple[bool, Optional[str]]`:
         - Try `subprocess.run(["difft", ...], capture_output=True, text=True)`
         - If difftastic not found (FileNotFoundError), return (False, None)
         - If found, return (True, stdout)
         - Use timeout of 10 seconds to prevent hanging on large files

    2. Update `src/kicad_agent/crossfile/__init__.py` barrel exports:
       - Add `structural_diff`, `DiffEntry`, `DiffResult`, `DiffType`
