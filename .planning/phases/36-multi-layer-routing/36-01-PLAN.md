    **Step 2: Add ViaSegment to bridge.py.**

    Add the ViaSegment frozen dataclass (per RESEARCH.md code example):
    ```python
    @dataclass(frozen=True)
    class ViaSegment:
        x: float
        y: float
        from_layer: str
        to_layer: str
        diameter: float
        drill: float
        net: str

        def to_sexpr(self, uuid_tag: str = "") -> str:
            parts = [
                "  (via",
                f"    (at {self.x:.4f} {self.y:.4f})",
                f"    (size {self.diameter:.4f})",
                f"    (drill {self.drill:.4f})",
                f'    (layers "{self.from_layer}" "{self.to_layer}")',
            ]
            if self.net:
                parts.append(f'    (net 0 "{self.net}")')
            if uuid_tag:
                parts.append(f"    (uuid {uuid_tag})")
            parts.append("  )")
            return "\n".join(parts)
    ```
