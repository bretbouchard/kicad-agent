# LTspice

LTspice .asc file parsing, simulation command injection, and waveform analysis.

The ltspice module integrates with LTspice for circuit simulation. It parses .asc schematic files, injects simulation commands (.tran, .ac, .dc, .op), and analyzes raw waveform output using spicelib.

::: volta.ltspice
    options:
      show_source: true
